Test Identification

NUnit uses two key attributes to identify tests, supported by several others.

  • TestFixtureAttribute marks a class that contains tests.
  • TestAttribute marks a test method.
  • SuiteAttribute is an older method of creating test suites, which is still used by some shops.
  • DescriptonAttribute provides a description for a test.
  • PropertyAttribute is used to associate arbitrary properties with a test.
