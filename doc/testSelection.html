Test Selection

NUnit a number of attributes to control which tests are executed in a partiular run.

  • IgnoreAttribute marks tests as temporarily excluded from the run.
  • ExplicitAttribute indicates that a test should only be run if directly selected.
  • CategoryAttribute is used to group tests for selection or exclusion.
  • PlatformAttribute is used to include or exclude tests on particular platforms.
  • CultureAttribute indicates that a test should be run only for certain culture settings.
