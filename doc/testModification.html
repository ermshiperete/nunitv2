Test Modification

This includes several attributes that modify how a test is executed.

  • ExpectedExceptioAttribute is used to indicate that a test should throw a particular exception.
  • SetCultureAttribute causes the test culture setting to be changed while running the test.
  • RequiredAddinAttribute marks a test or an assembly as needing a particular addin.
